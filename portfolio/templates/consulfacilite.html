Consul Facilite | Washing Machine Intuitive washing settings

My role
Graphic design, UI/UX design
Team
Whirlpool - Industrial Design (Brazil)

Project

Consul Facilite Washing Machine was designed to be helpful during all the washing steps.Visiting consumers we observed that the main criteria to separate clothes before washing are: color (dark, light, colorful) and dirtiness (slightly dirty, dirty, very dirty). Therefore we created programs following the same logic. In order to be more intuitive, the programs were placed around a central knob.Yet the interface guides users through sequential settings. Finally an orange ruler inside the washing machine matches orange scales around the knob and inside the drawer; this helps users to set the proper volume of water, detergent and fabric softener.
